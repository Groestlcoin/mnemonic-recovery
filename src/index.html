Testnet Seed Savior: Mnemonic Phrase Recovery Tool

v0.1

This tool is meant to help users with recovering a slightly incorrect Groestlcoin Testnet mnemonic phrase (AKA backup or seed). You can enter an existing BIP39 mnemonic and get derived adresses in various formats. If a word is wrong, the tool will try to suggest the closest option. If a word is missing or unknown, please type "?" instead and the tool will find all relevant options.

Enter your mnemonic

example:"phrase brief ceiling dream rack install fault insane panic surround glory ? library brother hill sauce access child notice picnic dinner panda purity poem"

The tool will suggest several options for the missing word and the relevant one will be "asset", with the following Groestlcoin Testnet address, listed in the "BIP44 GRS Testnet Address" column "n1wjsAdenUarSVgZNPLPcaUmVs6nhx6XXr"

Offline Usage

You can use this tool without having to be online.

In your browser, select file save-as, and save this page as a file.

Double-click that file to open it in a browser on any offline computer.

Alternatively, download the file from the repository - https://github.com/Groestlcoin/mnemonic-recovery/tree/testnet
